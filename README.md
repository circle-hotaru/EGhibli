# EGhibli

EGhibli is a simple online Ghibli information platform.

## Demo

View the application at [EGhibli](http://eghibli.mightyherox.me/).

## Tech Stack

### Font-end

- Vue
- Vue Router
- Vuex
- Vue-lazyload
- Bootstrap
- ElementUI

### Back-end

- SpringBoot
- Maven
- MyBatis
- MySQL
- Lombok
